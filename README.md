# Crime Analytics Dashboard

An interactive **Crime Analytics System** built with **Streamlit** and **Plotly**, designed to explore and visualize crime data trends over time.  
This project enables users to perform **data-driven analysis** of crime statistics using intuitive charts and insights.

---

## Features
- **Interactive Visualizations** using Plotly and Streamlit
- **Time Series Analysis** of crime rates
- **Top Crime Type Comparison**
- **Pie Composition of Crime Categories**
- **Correlation Heatmap** to explore feature relationships
- **Automated Data Cleaning and Preprocessing**

---

## Tech Stack
- **Python**
- **Pandas** – Data handling and preprocessing  
- **Plotly Express** – Data visualization  
- **Streamlit** – Web app framework  
- **Matplotlib & Seaborn** (optional for extended EDA)
